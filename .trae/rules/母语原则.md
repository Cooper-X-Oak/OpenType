# 母语原则 (Native Language Principle)

## 核心规则

- **我的母语是中文**：所有沟通、文档、代码注释、Commit Message 必须优先使用中文。
- **思考与回复**：必须以中文为第一语言进行思考和回复（除非用户明确要求英文）。
- **OpenSpec 规范**：所有 OpenSpec 工件（Proposal, Specs, Design, Tasks）必须使用中文编写，专有名词除外。
- **Todo 列表**：任务列表内容必须使用中文。
- **专业术语**：保留专业术语的英文原文（如 OpenSpec, Next.js 等），避免过度翻译。
